Introducción
El análisis exploratorio de datos es una etapa fundamental para comprender la información contenida en un conjunto de datos. En este proyecto se desarrolla una aplicación web con Streamlit que permite visualizar datos de anuncios de vehículos mediante gráficos interactivos. A través de esta aplicación, el usuario puede explorar la distribución del kilometraje y analizar la relación entre el kilometraje y el precio, lo que facilita la interpretación de la información y el aprendizaje del análisis de datos.
La realización de este proyecto es importante porque permite  aplicar conceptos básicos de análisis de datos y programación en un entorno práctico. Lo que ayuda a comprender mejor los datos y a desarrollar habilidades digitales, como el uso de Python, el análisis exploratorio de datos y la creación de aplicaciones web.
En conclusión, la aplicación web desarrollada cumple con el objetivo de facilitar el análisis exploratorio de datos de vehículos mediante gráficos interactivos. Gracias al uso de Streamlit y Plotly Express, es posible visualizar información clave de manera sencilla y dinámica, lo que permite identificar patrones y relaciones entre las variables analizadas. Este proyecto contribuye al desarrollo de habilidades tecnológicas y al fortalecimiento del pensamiento analítico.
